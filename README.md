## 🚀 Self-Driven Upskilling: What I Built

### 🛒 AI-Powered E-Commerce Platform
An 8-service microservices architecture demonstrating enterprise-scale system design.
- **Services:** Eureka Discovery, API Gateway, User, Product, Cart, Order, Payment, AI
- **Stack:** Java 21, Spring Boot 3.2.5, Spring Cloud, LangChain4j
- **AI/Data:** Redis Stack as a **vector database** for semantic product search & recommendations
- **Databases:** Oracle, MongoDB
- Focus: service discovery, inter-service communication, and AI-augmented commerce workflows

### 🔍 AI-Based Fraud Detection System
A real-time fraud scoring pipeline combining full-stack engineering with applied ML.
- **Frontend:** React + Vite

- **AI Microservice:** Python (Flask) + scikit-learn Random Forest model
- **Database:** PostgreSQL

### 💳 Financial Transaction Management App

- **Backend:** Spring Boot, **JWT-secured API Gateway** with edge-level request validation
- **AI Agent:** LangChain4j + OpenAI GPT-4o-mini for transaction insights and Q&A
- **Frontend:** React + TypeScript
- **Database:** PostgreSQL
- **Quality:** JUnit/Mockito backend test suite, React Testing Library frontend tests
- **Real-time:** Server-Sent Events (SSE) for live transaction status streaming to the client
